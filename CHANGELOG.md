# Changelog

- Filter support: `--state`, `--label`, `--assignee`, `--repo`, `--owner`
- `/tracker:doctor` diagnostic command with gh CLI, auth, and repo scope checks
- Repo scope config via `.local.md` (heurema org ~15 repos allowlist)
